# Docker 常用命令





* 拉取镜像

``` linux
docker pull 镜像名称:版本号
docker pull mysql:5.6
```

* 运行镜像

``` linux
docker run 
```

* 复制文件

``` linux
docker cp [OPTIONS] CONTAINER:SRC_PATH DEST_PATH|-

docker cp /www/runoob 96f7f14e99ab:/www/
```

* 进入容器

``` linux
docker exec -it c26fb39cf7e8 bash
```
